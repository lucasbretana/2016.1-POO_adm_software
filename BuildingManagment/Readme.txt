Diretórios:
src/
  Código fonte

doc/
  Documentação gerados pelo JavaDoc

bin/
  Binários, executáveis, .jar e .class

extra/
  Bills/
    Pasta padrão para arquivo de contains
  Residents/
    Pasta padrão para arquivo de residentes
  Output/
    Pasta contendo dados financeiros do prédio.

Get Started:
java -jar app.jar para executar o programa
Uma vez dentro do programa selecione a opção desejada do menu.
  Menu:
    0 -- Sair do programa
    1 -- Passar arquivo de dados do morador apartir do diretório padrão sem extenção de tipo
    2 -- Passar arquivo de dados do morador apartir do diretório padrão sem extenção de tipo
    3 -- Listar moradores por andar
    4 -- Listar moradores por apartamento(é permitido mais de um morador em cada apartamento)
    5 -- Listar moradores com garagem
    6 -- Passar arquivo de contas do prédio apartir do diretório padrão sem extenção de tipo
          Arquivo com o nome do mês será gerado em extra/Output/ contendo dados financeiros do prédio
